EDA Project by 장치근

Dataset : https://www.kaggle.com/spscientist/students-performance-in-exams

1. 성별에 따른 각 점수

- 사용한 plot : Boxplot
- x축 : male (남성), female (여성)
- y축 : 글쓰기 점수, 읽기 점수, 수학 점수

결론

- 평균적으로 남자보다 여자는 수학점수가 높다.
- 여자는 남자보다 읽기, 쓰기점수가 높다.

2. 학위에 따른 평균 점수

- 사용한 plot : Boxplot
- x축 : degree(학위)
- 학위 순서
       - 'some high school' < 'high school' < 'some college' < "associate's degree" < "bachelor's degree" < "master's degree"
- y축 : mean = (math + writing + reading)/3 (평균점수)

결론

- 최종 학위가 더 높을수록 평균점수가 점점 더 높아진다.

3. 점심식사 여부에 따른 평균 점수

- 사용한 plot : violinplot
- x축 : standard (든든~), free/reduced (부실..)
- y축 : mean = (math + writing + reading)/3 (평균점수)

결론

- 역시 든든하게 먹고 온게 점수 향상에 도움이 된다!
