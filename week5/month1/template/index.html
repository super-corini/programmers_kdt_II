YEAR



x축은 year을 의미하며, y축은 해당 년도에 생긴 terror의 수를 의미합니다. 1993년의 데이터는 존재하지 않았으며, 경향성을 봤을 때 year이 증가할 수록 terror의 수가 증가 즉, 비례하는 것을 알 수 있습니다.

Country Code & Year



해당 데이터에는 각 나라마다 country code가 존재하고, 이는 나라의 순서대로 되어있습니다. 그래프를 보면 x축은 똑같이 year이지만, y축은 country code를 평균 낸 결과입니다. 아까와는 반대로 예전의 테러들이 country code가 높은 쪽에서 나타났다는 것을 알 수 있습니다.

Region Code & Extended



x축은 region code로 동남아시아, 남아프리카 등 세계를 여러 지역으로 나눈 것을 뜻합니다. y축은 extended로 해당 terror가 24시간 이상 지속되었는지 즉, 장시간 테러인지 아닌지를 판별합니다. 장시간 테러가 일어났으면 1, 아니면 0으로 평균을 낸 결과 region code 3,6,7,11 이 장시간 테러가 날 확률이 높다는 것을 알 수 있습니다.

Target & Extended



해당 그래프는 HeatMap입니다. 여기서 y축은 target을 나타내고, extended는 장시간테러 여부를 말합니다. 이 graph의 의미는 target에 따라 장시간 테러를 할 확률이 달라진다는 것입니다. 만약 테러의 타겟이 NGO, Maritime, Journalists, Tourist라면 장시간 테러가 일어날 확률이 높다는 것입니다.
